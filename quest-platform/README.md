# Платформа квестов ? портфолио (аналитика)

Учебный кейс: концепция продукта, требования, модель данных, SQL (рекурсивная валидация графа квеста).

## Навигация по артефактам

| Раздел | Содержание |
|--------|------------|
| `docs/` | Vision & Scope, BRD, пользовательские и системные требования, функциональные требования, глоссарий, сводное ТЗ |
| `business_processes/` | Use Case: создание квеста, прохождение, валидация графа |
| `data_modeling/` | DDL `schema.sql`, словарь данных |
| `sql_queries/` | Рекурсивный CTE для проверки графа (`validation_cte.sql`) |

## Воспроизведение БД (PostgreSQL)

```text
psql -d your_db -f data_modeling/schema.sql
psql -d your_db -f sql_queries/validation_cte.sql
```

При необходимости добавьте `test_data.sql` отдельно, если появится в проекте.
