# Vision & Scope — платформа квестов

## 1. Концепция продукта

**Продукт:** веб-платформа для создания, публикации и прохождения сюжетных квестов с ветвлениями (граф состояний).

**Ценность для автора:** быстро собрать квест из шагов и переходов, опубликовать, получать статистику прохождений.

**Ценность для игрока:** проходить квест по сценарию, видеть последствия выборов.
