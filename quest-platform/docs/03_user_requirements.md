# Пользовательские требования (роли и сценарии)

## 1. Роли

| Роль | Описание | Основные цели |
|------|----------|----------------|
| **Гость** | Неавторизованный пользователь | Просмотр каталога (если разрешено), регистрация / вход |
| **Игрок** | Зарегистрированный пользователь | Прохождение квестов, сохранение прогресса |
| **Автор** | Создатель контента | Создание и редактирование квестов, публикация |
| **Модератор / Админ** | Управление платформой | Валидация контента, блокировки, настройки *(уточните)* |
