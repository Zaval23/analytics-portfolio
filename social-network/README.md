# Social network — учебная база данных (PostgreSQL)

Портфолио: реляционная модель социальной сети, DDL, тестовые данные и **60 SQL-запросов** по техническому заданию.

## Структура репозитория

| Путь | Назначение |
|------|------------|
| `docs/ТЗ_соцсеть.md` | Формулировки задач и глоссарий |
| `docs/ER_diagram.png` | ER-диаграмма (экспорт из Erwin или другого инструмента) |
| `sql/schema.sql` | Создание объектов БД (`CREATE TABLE`, ограничения, индексы) |
| `sql/test_data.sql` | Заполнение тестовыми данными (**порядок вставок** соблюдайте из-за FK) |
| `sql/all_queries.sql` | Все 60 запросов одним файлом (нумерация совпадает с ТЗ) |
| `sql/advanced_queries.sql` | 10 самых сложных или показательных запросов |

## Как воспроизвести локально

1. Установите PostgreSQL (рекомендуется **14+**).
2. Создайте базу, например: `CREATE DATABASE social_network;`
3. Выполните файлы **строго по порядку**:
   - `schema.sql`
   - `test_data.sql`
   - при необходимости — запросы из `all_queries.sql` / `advanced_queries.sql`

Либо из `psql`:

```text
\i sql/schema.sql
\i sql/test_data.sql
\i sql/all_queries.sql
```
